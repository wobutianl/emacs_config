#+TITLE: tableau认识

* 初识软件
** 软件是做什么的？
- 数据分析软件
- 自助式分析
- 大数据
- 与DB自动连接
- 主打（一键可视化）
- 具体
  - 匹配最佳图表
  - 地统计（与地图绑定）
  - 数据引擎
  - 元数据管理
  - VizQL，鼠标代替DB检查。
** 有什么同类软件？
- Excel + Open Office Calc
- TIBCO Spotfire：部署网络分析应用
- QlikView + Grapheur：交互式商业智能工具（数据挖掘）
- Gephi：网络（免费）
- Visokio Omniscope：多标签，多视图
- Pauopticon：做最好的可视化实践
- 其余：Wolfram Mathematics + Matlab + Plot + Spss + Stata + VisualizeFree + Dundas + Wonder graphs + ggplot + data graph + Orange 
** 软件与其他同类软件的不同之处？
- 大数据
- 动态的非代码的DB查询
- 一键式的可视化
** 面向的人群是怎么样的？
- 大公司，有大数据需求，
- 对图表统计有行业要求的
- 数据变化大的公司。
