#+TITLE: 地图学


* 想从本书学到什么
- 地图学是做什么的？
- 主要内容
- 有哪些基本概念？
- 主要想知道的点：坐标系，投影，概括

* 地图组成
- 地图图形：用地图符号表示制图区域
- 数学要素：位置分布，几何精度（投影，坐标，比例尺，控制点）
- 辅助要素：地图说明（时间，图名。。。）

* 作用
- 适应人的图形感受
- 空间信息载体，积累器
- 客观世界的模型

* 测绘地图
- 地面点：地理坐标系，平面坐标系，高程。。
- 大地控制网：大面积地图测绘时，对制图区域的分幅测量
- 地球椭球体：模拟地球形状的一个椭球

* 地图投影
- 概念：地球椭球体上的经纬网，按一定的数据法则转移到平面上的过程
|        |        |        |
|--------+--------+--------|
| 椭球体 | 经纬网 | 坐标系 |
| 投影法 | 平面坐标   | 直角坐标系 |
椭球 -> 平面 ------ 投影

- 投影选择
  - 世界范围：正圆柱，伪圆柱，多圆锥
  - 墨卡托：世界航线图，世界交通图，世界时区图
  - 半球体：横轴方位，斜轴方位
  - 图家：方位投影

* 地图符号
- 概念：表达地图内容，事务空间位置，形状，质量，数量的。
- 构成：形状，尺寸，颜色
- 分类：｛点，线，面
         定位符，说明符｝
- 表示移动要素：箭头
- 表示统计：统计图
- 地形要素：等高线，分层设色，阴影（晕渲法）
- 注记：字体，大小，颜色，排列，配置

* 地图概括
- 概念：制图区域客观事务的取舍与简化
- 包括：内容取舍，数量化简，质量化简，形状化简
- 考虑：比例尺，地图用途，主题，地理
