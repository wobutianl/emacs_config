* 组成
- 用例图（use-case）:(参与者（小人） + 用例）
  - 继承关系， 扩展关系， 包含关系
- 类图（class）
  - +，－，# :: 公有，私有，保护
  - 关联，泛化，实现，多重 :: 成员变量，继承（依赖），实现抽象函数，多对多
  - 依赖  < 关联 < 聚合 < 组合
  - 依赖与关联的区别（其实没必要弄这么清楚呀……）
- 状态图 :: 实体在时间流过程中，处于的可能的状态
- 时序图
  - 生命线 + 消息 + 激活
  - 生命线 :: 这个实体存在的时间长
  - 消息 :: 同步，异步，返回
  - 激活 :: 实体开始执行（矩形框，代表运行的时长）
- 协作图（用得不多）
- 活动图 ＝＝ 流程图
- 构件图 :: 各组件的依赖关系 ＝ 包 + 构件
- 部署图（用得不多）
